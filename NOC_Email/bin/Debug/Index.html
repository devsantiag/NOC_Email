Introdução Instalação Como Usar Configurações Envio de E-mails Suporte

1. Introdução

O sistema NOC_Email foi desenvolvido para facilitar o envio de e-mails automatizados via Outlook e gerenciar dados de clientes usando arquivos locais. Este manual explica como instalar, configurar e utilizar o sistema.

2. Instalação

Basta copiar a pasta do executável para o computador. Certifique-se de que o Outlook esteja instalado e configurado corretamente no sistema.

3. Como Usar

Abra o executável NOC_Email.exe. A interface principal permitirá selecionar um cliente, visualizar seus dados e enviar e-mails com base nos modelos configurados.

Validação de Campos Vazios

Para garantir a integridade dos dados e evitar o envio de e-mails incompletos ou o salvamento de registros inválidos, o sistema NOC_Email utiliza validações automáticas em todos os campos essenciais da interface.

Cada campo de entrada — como Razão Social, Expediente, E-mail, Telefone e Tipo de Defeito — entre outros, deve estar corretamente preenchido antes que o sistema permita salvar os dados ou enviar e-mails.

Se um campo obrigatório estiver vazio ou contiver apenas espaços em branco:

  • A ação é automaticamente bloqueada.
  • Uma mensagem de alerta é exibida: "Ação Bloqueada! Por favor, preencha todos os campos obrigatórios antes de enviar o e-mail."
  • O usuário é orientado a preencher corretamente o campo.

Antes de clicar em Salvar nas configurações ou Enviar E-mail no formulário principal, verifique se todos os campos estão:

  • Preenchidos corretamente (sem espaços em branco apenas).
  • Com informações válidas (principalmente no campo de e-mail e telefone).
  • Sem campos deixados em branco acidentalmente.

Isso garante um funcionamento seguro do sistema e evita falhas na comunicação com o cliente.

4. Configurações

Use o botão "Config ..." para cadastrar informações específicas, como e-mail, razão social e entre outras. Os dados serão salvos automaticamente em arquivos de texto.

Para deletar uma pasta por completo via comando, basta selecionar um campo de texto, por exemplo 'Razão Social' e digitar: :empty all razao social.

Comandos do Sistema - Exclusão de Dados

:empty all razao social
Comando usado para apagar todos os dados salvos da razão social. O sistema pede confirmação do usuário e, se confirmada, limpa os registros e o campo do ComboBox correspondente.
:empty all expediente
Comando utilizado para apagar todos os dados salvos do expediente do cliente. Após a confirmação, os dados são excluídos e o campo de exibição é limpo.
:empty all emails
Comando para apagar todos os e-mails salvos. O sistema solicita confirmação do usuário antes de excluir definitivamente os registros.
:empty all telefone
Comando usado para apagar todos os dados salvos relacionados a números de telefone. Exige confirmação prévia do usuário.
:empty all tipo defeito
Comando para apagar todos os registros salvos de tipo de defeito. Assim como os demais, requer confirmação do usuário para evitar exclusões acidentais.

Exclusão de Dados Individuais

Além dos comandos que excluem todos os dados de uma categoria, o sistema NOC_Email também possui um botão exclusivo para remover apenas um único item selecionado no ComboBox, sem afetar os demais registros.

Para utilizá-lo:

  • Selecione o item desejado no ComboBox correspondente (ex: razão social).
  • Clique no botão "Excluir".
  • O sistema removerá apenas o dado selecionado, mantendo os outros intactos.

Essa função é ideal para gerenciar os registros individualmente, sem necessidade de limpar todos os dados salvos.

5. Envio de E-mails

Com os dados carregados, clique no botão de envio para utilizar o Outlook e disparar o e-mail com os campos preenchidos. Verifique se o Outlook está aberto e autenticado.
