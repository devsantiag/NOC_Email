Manual do Sistema NOC_Email

Versão 1.0 | Desenvolvido por Fabiano Santiago

Introdução Instalação Como Usar Configurações Envio de E-mails Suporte

1. Introdução

O NOC_Email é uma aplicação Windows Forms desenvolvida para automatizar o envio de e-mails operacionais a partir de modelos pré-configurados. Seu objetivo principal é padronizar a comunicação entre os setores técnicos e clientes, utilizando integração com o Microsoft Outlook e dados salvos em arquivos de texto locais.

2. Instalação

Para utilizar o sistema:

  • Copie a pasta do executável NOC_Email.exe para o computador desejado.
  • Certifique-se de que o Outlook está instalado, configurado e com uma conta autenticada.
  • Não é necessária instalação adicional — trata-se de uma aplicação portátil.
  • Crie a pasta /data/ na raiz do executável, se ainda não existir.

3. Como Usar

Ao abrir o sistema:

  • Selecione um cliente no ComboBox da tela principal.
  • As informações cadastradas aparecerão automaticamente nos campos (razão social, e-mails, tipo de defeito etc.).
  • Clique em “Enviar” para preparar o e-mail com base no modelo.

O conteúdo será preenchido automaticamente e exibido em uma nova janela do Outlook, permitindo revisão antes do envio.

4. Configurações

O botão “Configurar Dados” permite que você cadastre ou edite os dados dos clientes. É possível configurar:

  • Razão Social
  • E-mails de Contato
  • Telefone
  • Tipo de Defeito

Esses dados são salvos em arquivos de texto na pasta /data/, com um arquivo por cliente. O nome do arquivo segue o padrão razao_social.txt.

5. Envio de E-mails

Ao clicar no botão de envio, o sistema:

  1. Carrega os dados salvos do cliente selecionado.
  2. Insere essas informações no modelo de e-mail.
  3. Aciona o Outlook via interop para abrir um novo e-mail pronto para envio.

Importante: o Outlook precisa estar aberto e com o perfil autenticado para que a automação funcione corretamente.
