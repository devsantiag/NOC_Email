## English

### Overview

The **NOC_Email** repository was designed to manage functionalities related to automatic e-mail sending within a NOC (Network Operations Center) monitoring context. Created in September 2024, it is public and uses C# as its base language.

**Project Status: Discontinued**  
This project has been discontinued. All features and content will be integrated into the new repository [NOC_Actions](https://github.com/devsantiag/NOC_Actions), where NOC monitoring and notification automation will be centralized and improved.

  Project overview file, typically containing basic usage instructions, project goals, or essential commands.

  Visual Studio solution file (C#), organizing internal projects for easier development, build, and maintenance.
- [`NOC_Email/`](https://github.com/devsantiag/NOC_Email/tree/main/NOC_Email)  
  Main project directory, containing source files, implementations, and specific resources.

### Main features

- Management and sending of automatic e-mails for network operation notifications.
- Modular structure ready for integration with third-party or internal NOC systems.
- Uses C# solution format (.sln) for maintainability and project scalability.
